# master_thesis
This repository contains all files used for the set up and analysis of my research: "Exploring the Effect of Uncertainty Indicators on Trust in AI generated Political Information: An Experimental Study using a Chatbot in the Context of German Migration Politics"

The four HTML files contain each version of the chatbot with; version A: no uncertainty indicator; version B: visual uncertainty indicator; version C: textual uncertainty indicator; version D: numerical uncertainty indicator. The chatbot uses the openAI API but the API key is not stored in the HTML files. For running the files, a key would need to be inserted. 

The survey_analysis.ipynb file contains the analysis of the survey data, including various statistical tests and corresponding assumption checks as well as visualisations. 

The csv file containing all survey responses is not included in this repository due to privacy concerns but (an anonymised version) can be made available upon reasonable request.
